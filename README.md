# Tuition Management System

## Project Name
Tuition Management System

## Purpose
A complete platform where students, tutors, and admins can manage tuition activities.

## Features
- Student: Post tuitions, manage applications, payments
- Tutor: Apply for tuitions, track earnings
- Admin: Manage users, tuitions, reports
- JWT Authentication
- Stripe Payment Integration
